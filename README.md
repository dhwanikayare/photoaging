Photoaging Burden Index (PBI)

AI-based photoaging detection using convolutional neural networks and environmental exposure modelling.

Project Overview

This project develops a prototype system for estimating photoaging burden by combining facial image analysis with lifestyle and environmental exposure factors.

Photoaging refers to extrinsic skin aging caused primarily by ultraviolet radiation and environmental exposures, which manifest clinically as wrinkles, pigmentation irregularities, and loss of skin elasticity.

The system integrates:

• Deep learning–based facial analysis
• Lifestyle questionnaire data
• Environmental pollution exposure

These components are combined into a Photoaging Burden Index (PBI) representing overall photoaging risk.
